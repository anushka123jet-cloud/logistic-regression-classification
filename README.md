# 🚀 Logistic Regression Classification

A complete Machine Learning classification project using **Logistic Regression** on a Scikit-learn dataset. This project demonstrates the end-to-end workflow of building a classification model, evaluating its performance using **Accuracy Score** and **Confusion Matrix**, and optimizing results through **GridSearchCV** and **RandomizedSearchCV** for hyperparameter tuning.

## ✨ Key Features
✔️ Logistic Regression Model Training  
✔️ Accuracy Score Evaluation  
✔️ Confusion Matrix Analysis  
✔️ Hyperparameter Tuning with GridSearchCV  
✔️ Hyperparameter Tuning with RandomizedSearchCV  
✔️ Performance Comparison and Best Model Selection  

## 🛠️ Tech Stack
**Python • NumPy • Pandas • Matplotlib • Seaborn • Scikit-learn**

## 📂 Workflow
```text
Dataset → Data Preparation → Train-Test Split → Logistic Regression
        → Accuracy & Confusion Matrix Evaluation
        → GridSearchCV Optimization
        → RandomizedSearchCV Optimization
        → Best Model Selection
```
